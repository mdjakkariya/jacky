Privacy by design

Nothing leaves your machine

Everything runs locally on your Mac. The only two off-device features are opt-in, off by default, and clearly disclosed.

Runs locally

Speech, the language model (Ollama), memory and actions all run on-device. No cloud account needed.

Permission gate

Every action is risk-classified. Read-only runs through; destructive asks first; all of it is audited.

Two opt-in exceptions

Web search sends only the query; optional Claude cloud mode sends the conversation — never audio, never actions.

What it does

One assistant, hands on your whole Mac

Chat or voice

Type in the chat drawer, or enable voice and say “jack, …” for hands-free — or push-to-talk.

App & web control

Open, focus, hide, quit apps; open a website; close a browser tab — by voice or text.

Find & open files

On-device Spotlight search. Pick a result and open it or reveal it in Finder, right from chat.

Acts through tools

The model plans; the permission gate executes. It never runs an action blindly — and it’s audited.

System & memory

Check battery, Wi-Fi and disk. Remembers your name and preferences — stored locally.

Local or cloud LLM

Fully local on Ollama by default, or switch to Claude. Your choice, clearly disclosed.

Get started

Download the dev preview

Grab the latest .dmg from GitHub Releases — a single, self-contained app. Voice models download on demand the first time you enable voice.

Download .dmg Read the docs
1 · Install

Drag Jack to Applications. First launch: right-click → Open (unsigned dev preview).

2 · Set up

Jack walks you through the LLM (Ollama or a Claude key) and macOS permissions.

3 · Talk

Type in chat, or enable voice and say “jack, what’s the time.”
