Free & open source · No account · Local LLM by default

Privacy by design

Nothing leaves your machine

Everything runs locally on your Mac. The few off-device features are opt-in, off by default, and clearly disclosed — and every off-device send is shown and confirmed first.

Runs locally

Speech, the language model (Ollama), memory and actions all run on-device. No cloud account needed.

Permission gate

Every action is risk-classified. Read-only runs through; destructive asks first; all of it is audited.

Opt-in exceptions

Web search, Claude cloud and remote MCP servers — opt-in, disclosed, and confirmed first.

What it does

One assistant, hands on your whole Mac

Chat or voice

Type in the chat drawer, or say “jack, …” for hands-free voice — or push-to-talk.

Controls your Mac

Open and quit apps, open sites, find & open files, check battery, Wi-Fi and disk.

Connect your tools

Plug in GitHub, Notion, Slack and more over MCP — gated like every built-in tool.

Remembers you

Learns your name and preferences to personalize replies — stored on-device, never uploaded.

Local or cloud LLM

Fully local on Ollama by default, or switch to Claude. Your choice, clearly disclosed.

Acts through tools

The model plans; the permission gate executes and audits. Nothing runs blindly.

Get started

Download the dev preview

Grab the latest .dmg from GitHub Releases — a single, self-contained app. Voice models download on demand the first time you enable voice.

Download .dmg Read the docs
1 · Install

Drag Jack to Applications. First launch: right-click → Open (unsigned dev preview).

2 · Set up

Jack walks you through the LLM (Ollama or a Claude key) and macOS permissions.

3 · Talk

Type in chat, or enable voice and say “jack, what’s the weather.”
