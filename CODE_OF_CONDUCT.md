# Code of Conduct

This project follows the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## The short version

Be respectful and constructive. We welcome contributions and discussion from
everyone, regardless of background or experience. Harassment, personal attacks,
discriminatory language, and other unprofessional conduct are not tolerated — in
issues, pull requests, or any other project space.

## Reporting

If someone is behaving unacceptably, report it privately to the maintainer
(**@mdjakkariya**) via a GitHub direct message or a private channel. Reports are
handled confidentially, and the maintainer may take any action they deem
appropriate, up to removing a contributor from the project.

For the full text and enforcement guidelines, see the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
